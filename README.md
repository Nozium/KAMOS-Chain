## Desc for Hackathon
### Technicality
> 取り組んでいる問題の複雑さ、またはその解決へのアプローチとは？

### Originality
> 新しい問題や未解決の問題に取り組んでいるか、既存の問題に対してユニークで創造的な解決策を生み出しているか？

### Practicality
> プロジェクトの完成度や機能性はどうか？想定される利用者が使用できる状態になっているか？

### Usability (UI/UX/DX)
> プロジェクトは使いやすいか？ユーザーとの摩擦をなくすための努力がなされているか？

### WOW factor
> 従来のプロダクトにはもたらし得なかった新しい視点はあるか？
